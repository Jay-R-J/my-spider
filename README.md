# 终极智能爬虫 - 个人百度蜘蛛

本项目是一个完全免费、自动运行的爬虫系统，部署在 GitHub Actions 上。它能从种子网址开始探索网页，智能提取标题、描述、正文、价格、评分等结构化数据，并将详细报告通过 QQ 邮箱发送给你。

## 功能特点
- 自动发现链接，持续扩展抓取范围
- 遵守 robots.txt
- 支持域名白名单
- 智能正文提取（三级降级）
- 结构化数据解析（JSON-LD、微数据）
- Open Graph 元数据提取
- 每日定时运行，进度持久化
- 邮件推送详细报告

## 配置方法
见仓库内各文件注释。

## 注意事项
请确保抓取行为合法合规，不要对目标网站造成压力。
